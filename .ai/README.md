# Antigravity Directory

This directory contains resources and generated artifacts for the Antigravity AI assistant.

## Directory Structure

- `plans/`: Implementation plans and architectural documents generated during the planning phase.
- `analysis/`: Research notes, architecture deep dives, and analysis results (e.g., performance audits, bug root cause analyses).
- `prompts/`: Custom prompt snippets or context instructions used to guide the assistant.
- `configurations/`: Settings, configuration files, and state-related data.
